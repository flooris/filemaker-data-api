Improvements:
* Upgraded to PHP 8.0
* Added Eloquent dependencies (Laravel) v9 support
* Added FmBase abstract classes (FmBaseObject, FmBasePortalObject and FmBaseRepository) for dev friendly API record consuming
* Tuned the Readme.md
